## Debug Logging

To assist with troubleshooting, you can enable debug logging for Chime TTS and capture detailed log messages. Follow the steps below:

1. Turn on debug logging for the Chime TTS integration:
   ```
   action: logger.set_level
   data:
     custom_components.chime_tts: debug
   ```

2. Call the `chime_tts.say` action again.

3. Open your log messages in Home Assistant:
   <a href="https://my.home-assistant.io/redirect/logs/?filter=chime_tts" alt="Show logs of Home Assistant" target="_blank"><img src="https://my.home-assistant.io/badges/logs.svg" /></a>

4. Enter `chime_tts` in the "Search logs" field and click `LOAD FULL LOGS`
