## 2. Home Assistant Forum
Chime TTS has an active community on the Home Assistant forum. The main [Chime TTS post](https://community.home-assistant.io/t/chime-tts-play-audio-before-after-tts-audio-lag-free/578430) is a great place to ask questions, search for answers, share experiences and get advice from other users.
