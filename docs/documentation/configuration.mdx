# Configuration

You can configure the following options for the [`chime_tts.say`](/chime_tts/docs/documentation/actions/say-action) and [`chime_tts.say_url`](/chime_tts/docs/documentation/actions/say_url-action) actions through the integration's configuration page:

<a href="https://my.home-assistant.io/redirect/integration/?domain=chime_tts" alt="Chime TTS integration" target="_blank"><img src={useBaseUrl('/img/documentation/show_chime_tts_on_my.png')} /></a>

### Timeout

The duration (in seconds) to wait before cancelling the [`chime_tts.say`](/chime_tts/docs/documentation/actions/say-action) or [`chime_tts.say_url`](/chime_tts/docs/documentation/actions/say_url-action) action calls.

<table>
<tr><td><b>Default Value:</b></td><td>`60`</td></tr>
<tr><td><b>Required:</b></td><td>Yes</td></tr>
</table>

:::tip

Increase the timeout duration intend to generate audio longer than 60 seconds in length.

:::

---

### TTS Audio Generation Timeout

The duration (in seconds) to wait before canceling a TTS audio generation attempt.

<table>
<tr><td><b>Default Value:</b></td><td>`30`</td></tr>
<tr><td><b>Required:</b></td><td>No</td></tr>
</table>

---

### Default TTS Platform

The default [`tts_platform`](/chime_tts/docs/documentation/actions/say-action/parameters#tts_platform) value to use when not specified.

<table>
<tr><td><b>Default Value:</b></td><td>None</td></tr>
<tr><td><b>Required:</b></td><td>No</td></tr>
</table>

---

### Default Language

The default [`language`](/chime_tts/docs/documentation/actions/say-action/parameters#language) value to use if the [`tts_platform`](/chime_tts/docs/documentation/actions/say-action/parameters#tts_platform) is set to the default TTS platform.

<table>
<tr><td><b>Default Value:</b></td><td>None</td></tr>
<tr><td><b>Required:</b></td><td>No</td></tr>
</table>

---

### Default Voice

The default [`voice`](/chime_tts/docs/documentation/actions/say-action/parameters#voice) value to use if the [`tts_platform`](/chime_tts/docs/documentation/actions/say-action/parameters#tts_platform) is set to the default TTS platform.

<table>
<tr><td><b>Default Value:</b></td><td>None</td></tr>
<tr><td><b>Required:</b></td><td>No</td></tr>
</table>

---

### Default Dialect

The default dialect domain ([`tld`](/chime_tts/docs/documentation/actions/say-action/parameters#tld)) value if the [`tts_platform`](/chime_tts/docs/documentation/actions/say-action/parameters#tts_platform) is set to Google Translate.

<table>
<tr><td><b>Default Value:</b></td><td>None</td></tr>
<tr><td><b>Required:</b></td><td>No</td></tr>
</table>

---

### Fallback TTS Platform

The [TTS platform](/chime_tts/docs/documentation/actions/say-action/parameters#tts_platform) to retry generating TTS audio if the specified/default platform failed.

<table>
<tr><td><b>Default Value:</b></td><td>None</td></tr>
<tr><td><b>Required:</b></td><td>No</td></tr>
</table>

---

### Default Offset

The default [offset](/chime_tts/docs/documentation/actions/say-action/parameters#offset) duration (in milliseconds) between chimes and TTS audio.<br />
Positive values introduce a delay, while negative values overlap the audio segments.

<table>
<tr><td><b>Default Value:</b></td><td>`450`</td></tr>
<tr><td><b>Required:</b></td><td>No</td></tr>
</table>

---

### Fade Transition

The duration (in milliseconds) to fade out the now playing audio _before_ an announcement, and then fade it back in _after_ the announcement is completed.

_This applies to [`chime_tts.say`](/chime_tts/docs/documentation/actions/say-action) action calls when the [`announce`](/chime_tts/docs/documentation/actions/say-action/parameters#announce) or [`fade_audio`](/chime_tts/docs/documentation/actions/say-action/parameters#fade_audio) parameters are used._

<table>
<tr><td><b>Default Value:</b></td><td>`500`</td></tr>
<tr><td><b>Required:</b></td><td>No</td></tr>
</table>

---

### Delay Before Removing Temporary Files

The delay duaration after [`chime_tts.say`](/chime_tts/docs/documentation/actions/say-action) playback is completed (in milliseconds), before removing the generated temporary audio file.

<table>
<tr><td><b>Default Value:</b></td><td>None</td></tr>
<tr><td><b>Required:</b></td><td>No</td></tr>
</table>

---

### Custom Chimes Folder

The local folder path containing custom audio files. Please [read here](/chime_tts/docs/documentation/actions/say-action/#custom-chimes-folder) for more information.

<table>
<tr><td><b>Default Value:</b></td><td>None</td></tr>
<tr><td><b>Required:</b></td><td>No</td></tr>
</table>

:::warning[Restart Requierd]

Any changes to the folder path or its contents will require a restart to take effect.

:::

---

### Downloaded Chimes Folder

The local folder path where downloaded chime MP3 files will be stored.

<table>
<tr><td><b>Default Value:</b></td><td>`{root}/media/sounds/temp/chime_tts/chimes/`</td></tr>
<tr><td><b>Required:</b></td><td>Yes</td></tr>
</table>

---

### Temporary MP3 Folder

The local folder path where generated MP3 files will be stored.

<table>
<tr><td><b>Default Value:</b></td><td>`{root}/media/sounds/temp/chime_tts/`</td></tr>
<tr><td><b>Required:</b></td><td>Yes</td></tr>
</table>

:::warning[Please Note]

Make sure the temporary folder path is inside a [media folder](https://www.home-assistant.io/integrations/media_source/) of your Home Assistant, otherwise the [`chime_tts.say`](/chime_tts/docs/documentation/actions/say-action) action will not be able to play the generated MP3 on your media_players.

:::

---

### `chime_tts.say_url` Folder

The publicly facing folder path where MP3 files generated by the [`chime_tts.say_url`](/chime_tts/docs/documentation/actions/say_url-action) action will be stored.

<table>
<tr><td><b>Default Value:</b></td><td>`{root}/config/www/chime_tts/`</td></tr>
<tr><td><b>Required:</b></td><td>Yes</td></tr>
</table>

---

### MP3 Cover Art

When enabled, adds the Chime TTS logo as the cover art in generated MP3 files.

<table>
<tr><td><b>Default Value:</b></td><td>`false`</td></tr>
<tr><td><b>Required:</b></td><td>No</td></tr>
</table>
