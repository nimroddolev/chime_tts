# Configuration

You can configure the following options for the [`chime_tts.say`](../documentation/actions/say-action) and [`chime_tts.say_url`](../documentation/actions/say_url-action) actions through the integration's configuration page:

<a href="https://my.home-assistant.io/redirect/integration/?domain=chime_tts" alt="Chime TTS integration" target="_blank"><img src={useBaseUrl('/img/documentation/show_chime_tts_on_my.png')} /></a>

### Timeout
Default Value: `60`

Set the duration (in seconds) to wait before terminating action calls.

---

### Default TTS Platform
Default Value: None

The default TTS platform to use when the [`tts_platform`](../documentation/actions/say-action/parameters#tts_platform) parameter is not specified in the action calls.

---

### Default Language ([Beta](../quick-start/installing-chime-tts#beta-versions))
Default Value: None

The default value for the [`language`](../documentation/actions/say-action/parameters#language) parameter to use when it is missing in the action calls.
This is only applied when the [`tts_platform`](../documentation/actions/say-action/parameters#tts_platform) is set to the default TTS platform.

---

### Default Voice ([Beta](../quick-start/installing-chime-tts#beta-versions))
Default Value: None

The default value for the [`voice`](../documentation/actions/say-action/parameters#voice) parameter to use when it is missing in the action calls.
This is only applied when the [`tts_platform`](../documentation/actions/say-action/parameters#tts_platform) is set to the default TTS platform.

---

### Default Offset
Default Value: `450`

The default offset (in milliseconds) to insert between chimes and TTS audio. Positive values introduce a delay, while negative values overlap the audio segments.

---

### Fade Transition
Default Value: `500`

The duration (in milliseconds) for fading out the currently playing audio before an announcement and fading it back in afterward. This applies only when the [`announce`](../documentation/actions/say-action/parameters#announce) or [`fade_audio`](../documentation/actions/say-action/parameters#fade_audio) parameters are used.

---

### Custom Chimes Folder
Default Value: None

The local folder path that contains custom audio files. Any changes to the folder path or its contents will require a restart to take effect. [More info](../documentation/actions/say-action/#custom-chimes-folder)

---

### Media Folder
Default Value: `local`

The name of your [media folder](https://www.home-assistant.io/integrations/media_source/#using-custom-or-additional-media-folders).

---

### Downloaded Chimes Folder
Default Value: `{root}/media/sounds/temp/chime_tts/`

The local folder path where downloaded chime MP3 files will be stored.

---

### Temporary MP3 Folder
Default Value: `{root}/media/sounds/temp/chime_tts/`

The local folder path where generated MP3 files will be stored.

---

### `chime_tts.say_url` Folder
Default Value: `{root}/config/www/chime_tts/`

The publicly facing folder path where MP3 files generated by the [`chime_tts.say_url`](../documentation/actions/say_url-action) action will be stored.

---

### MP3 Cover Art ([Beta](../quick-start/installing-chime-tts#beta-versions))
Default Value: `false`

Include the Chime TTS logo as cover art in the generated MP3 files.
