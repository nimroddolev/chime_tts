# Configuration

You can configure the following options for the [`chime_tts.say`](../documentation/services/say-service) and [`chime_tts.say_url`](../documentation/services/say_url-service) services through the integration's configuration page:

<a href="https://my.home-assistant.io/redirect/integration/?domain=chime_tts" alt="Chime TTS integration" target="_blank"><img src={useBaseUrl('/img/documentation/show_chime_tts_on_my.png')} /></a>

## Configurable Options

### Timeout
| Description | Default Value |
|-|-|
| Set the time (in seconds) to wait before service calls are terminated. | `60` |

### Default TTS Platform
| Description | Default Value |
|-|-|
| The TTS platform to use if service calls are missing the [`tts_platform`](../documentation/services/say-service/parameters#tts_platform) parameter. | *The first TTS platform configured in your Home Assistant instance.* |

### Default Offset
| Description | Default Value |
|-|-|
| The default offset (in milliseconds) to insert between chimes & TTS audio. Positive values will add a delay and negative values will overlay audio segments on top of each other. | `450` |

### Custom Chimes Folder
| Description | Default Value |
|-|-|
| A local folder path containing custom audio files. Any changes to the folder path or its file contents require a restart. [More info](../documentation/services/say-service/#custom-chimes-folder) | None |

### Media Folder
| Description | Default Value |
|-|-|
| The name of your [media folder](https://www.home-assistant.io/integrations/media_source/#using-custom-or-additional-media-folders). | `local` |

### Downloaded Chimes Folder
| Description | Default Value |
|-|-|
| The local folder path to store downloaded chime mp3 files. | `{root}/media/sounds/temp/chime_tts/` |

### Temporary MP3 Folder
| Description | Default Value |
|-|-|
| The local folder path to store the generated mp3 files. | `{root}/media/sounds/temp/chime_tts/` |

### `chime_tts.say_url` Folder
| Description | Default Value |
|-|-|
| The folder path to store MP3 files generated by the [`chime_tts.say_url`](../documentation/services/say_url-service) service. | `{root}/config/www/chime_tts/`

### MP3 Cover Art
| Description | Default Value |
|-|-|
| Include the Chime TTS logo as cover art in generated MP3 files | `false`
