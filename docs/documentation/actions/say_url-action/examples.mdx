# `chime_tts.say_url` Examples

The `chime_tts.say_url` action passess all arguments to the [`chime_tts.say`](/chime_tts/docs/documentation/actions/say-action) action _(with the exception of all audio playback parameters such (as media_player targets and other playback-related parameters, as there is no audio playback)_.

Please refer to the [`chime_tts.say` examples documentation](/chime_tts/docs/documentation/actions/say-action/examples) for a list of examples.
