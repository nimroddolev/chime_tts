# `chime_tts.say_url` Parameters

The `chime_tts.say_url` action accepts the exact same parameters as the [`chime_tts.say`](../say-action) action _(with the exception of all audio playback parameters such as media_player targets and other playback-related parameters, as there is no audio playback)_.

Please refer to the [`chime_tts.say` parameters documentation](../say-action/parameters) to learn more about how the parameters are used.
