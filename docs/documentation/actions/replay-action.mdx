# `chime_tts.replay` Action

The `chime_tts.replay` action calls the [`chime_tts.say`](/chime_tts/docs/documentation/actions/say-action) action with the exact same set of parameter values used in the previous [`chime_tts.say`](/chime_tts/docs/documentation/actions/say-action) action call.

## Parameters
There are no parameters to set for this action.

## YAML Example

```
action: chime_tts.replay
data:
```
