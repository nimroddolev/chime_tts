# Chime TTS Say Action

There are 4 actions added by Chime TTS:

1. The [`chime_tts.say`](./say-action) action generates a local MP3 file comprised of chime/s and/or TTS audio, for playback.

2. The [`chime_tts.say_url`](./say_url-action) action returns local/remote filepaths to files generated by [`chime_tts.say`](./say-action), without playback.

3. The [`chime_tts.replay`](./replay-action) action repeats the last action call to [`chime_tts.say`](./say-action) with the previous parameter values.

4. The [`chime_tts.clear_cache`](./clear_cache-action) action selectively removes cached audio files.
