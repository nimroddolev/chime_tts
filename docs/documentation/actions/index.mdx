# Chime TTS Actions

Chime TTS adds 4 actions for you to use in Home Assistant:

1. **[`chime_tts.say`](./say-action)** generates and plays a local MP3 file comprised of chime/s and/or TTS audio.

2. **[`chime_tts.say_url`](./say_url-action)** generates a local/remote MP3 file from [`chime_tts.say`](./say-action) without playback.

3. **[`chime_tts.replay`](./replay-action)** repeats the last [`chime_tts.say`](./say-action) action, using the same parameter values.

4. **[`chime_tts.clear_cache`](./clear_cache-action)** selectively deletes cached audio files from your file system.
